    public static getGeohashActions(currentGeohashes: Array<string>, newGeohashes: Array<string>): Object source

    Returns the geohash add, update and delete actions based on existing and new geohashes. This function guarantes that the event listeners get triggered correctly.

    e.g. currentGeohashes = ["a", "b", "c"] and newGeohashes = ["b", "c", "d"] would return add: ["d"], update: ["b", "c"], delete: ["a"]

    Params:

    NameTypeAttributeDescription
    currentGeohashes Array<string>

    The geohashes based on the current location.
    newGeohashes Array<string>

    The geohashes based on the new location.

    Return:

    Object

    An object containing the geohashes for the add, update and delete action.

    public static getGeohashesInBoundaries(boundaries: Object, precision: number): Array<string> source

    Returns all the visible Geohashes within the given Boundaries (also overlapping geohashes) The geohash length will be calculated from the given precision.

    This function works as a preselection for the entries and protects tons of them from running through an unnecessary query.

    Params:

    NameTypeAttributeDescription
    boundaries Object

    The boundaries object which builds the rectangle query borders.
    precision number

    The precision of the geohashes to return.

    Return:

    Array<string>

    An array containing all the geohashes within the prvided boundaries.
