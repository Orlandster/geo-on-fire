    public isInBoundaries(location: Object, boundaries: Object): boolean source

    Checks if the given location is within the boundaries This needs to be done, since getGeohashesInBoundaries() returns also overalpping locations, which our outside of the boundaries. The getGeohashesInBoundaries() will be used to reduce the number of possible locations, which provides us a huge performance gain.

    Params:

    NameTypeAttributeDescription
    location Object

    holds the current location
    boundaries Object

    holds coordinates of the boundaries

    Return:

    boolean

    Says if location is in or outside of boundaries
