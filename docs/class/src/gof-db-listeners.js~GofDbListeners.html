    public static attachDbListeners(type: string, dbRef: Object, geohashes: Array<string>, event: Object): Array<Object> source

    Attaches the database listeners based on the event type. The firebase event "child_moved" gets not supported.

    Params:

    NameTypeAttributeDescription
    type string

    The event type: "value", "child_added", "child_changed", "child_removed".
    dbRef Object

    The dbRef object containing the entries and locations path.
    geohashes Array<string>

    The geohashes based on the active query.
    event Object

    The query event object.

    Return:

    Array<Object>

    An array containing the listeners reference.
