    public createEntry(entry: Object, priority: number, maxPrecision: number): Promise<Object> source

    Creates a new entry in the database and sets all the according locations. (geohashes) priority and maxPrecison are the geohash lengths. e.g. (prio of 4 == abcd).

    Low priority (1-4) can slow up the speed on apps with massive scaling.

    Params:

    NameTypeAttributeDescription
    entry Object

    The new entry to create.
    priority number
    • optional
    • default: 5

    The priority of the new entry. (1 = high prio, 5 = low prio)
    maxPrecision number
    • optional
    • default: 9

    The maximum geohash precision. (7 - 9)

    Return:

    Promise<Object>

    A promise which is fulfilled once the entry was created.

    public getLocationsByBoundaries(boundaries: Object, startAt: number, endAt: number): function source

    Sets the active query object with all locations from given boundaries(south-west / north-east). The reulst can be limited with startAt and endAt. e.g. 1 - 50

    Needs to be called in a function chain with .on() or .once()

    Params:

    NameTypeAttributeDescription
    boundaries Object

    Boundaries object which builds the rectangle query borders.
    startAt number

    • default: 1

    The entry to start at - e.g. start for pagination.
    endAt number
    • optional
    • default: 50

    The etnry to end at - e.g. end for pagination.

    function

    The function itself to make .on() and .once() chains possible.
